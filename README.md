Scrapped flags from [https://r74n.com/pixelflags/](https://r74n.com/pixelflags/), along with their RGB and CIELAB pixel arrays

All the data can be downloaded from [data.zip](https://github.com/d1vij/images-and-pixel-data-for-flags/releases/download/v2.0.0/data.zip)

## Structure

```
data
├── catagories              // Files with arrays of flags grouped by catagory
│   ├── list.json           // Array of catagory names
│   ├── ...
├── flags
│   ├── all_flags_rgba.json // All flag's RGBA in a single file
│   ├── flag_urls.json      // URLS of flag images
│   ├── images/             // Images of flags
│   ├── lab/                // Flag pixel data as CIE-Lab
│   ├── rgb/                // Flag pixel data as RGBA
└── page.html
```
